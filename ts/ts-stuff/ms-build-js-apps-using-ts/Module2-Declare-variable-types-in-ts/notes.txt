[+] Types
=========================================================================================
    - All types are subtypes of top type called any type. Any type can represent any 
      js value
    - Types break into following categories:
        -> Primitive types:
            - boolean
            - number
            - string
            - enum -> user defined
            - void -> indicates absence of val. Used in func with no return value
            - null -> subtype of all other types
            - undefined -> subtype of all other types

        -> Object types:
            -> class
            -> interface
            -> array
            -> literals
        -> Type parameters
