## c++
```
g++ -std=c++ filename.cpp -o N
```
## java
- Note java filename can't have underscore and filename must match classname
```
java filename.java
```
## Python
```
python filename.py
```
## Typescript
```
ts-node filename.ts
```
## C#
- .cs file. Not a project
```
- csc filename.cs
- mono filename.exe
```
